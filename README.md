# 🚦 AI Smart Traffic Intelligence System

> AI-powered traffic monitoring system built using **Python, OpenCV, and YOLO** for real-time vehicle detection, traffic density analysis, vehicle counting, and adaptive traffic signal recommendations.

---

## 📖 Overview

The **AI Smart Traffic Intelligence System** is a Computer Vision project that demonstrates how Artificial Intelligence can be applied to intelligent traffic management.

The system detects vehicles in real time, counts them, analyzes traffic density, and recommends traffic signal timings based on traffic conditions.

This project was developed after completing my **Computer Vision Training**, where I applied the concepts learned into a real-world AI application.

---

## ✨ Features

- 🚗 Real-Time Vehicle Detection using YOLO

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/balodiyasanjay9/AI-Smart-Traffic-Intelligence-System.git
```

Move into the project folder

```bash
cd AI-Smart-Traffic-Intelligence-System
```

Install the required libraries

```bash
pip install -r requirements.txt
```

---

## ▶️ Run the Project

Open the Jupyter Notebook

```bash
jupyter notebook
```

Run

```text
AI_traffic_CV_linkedin.ipynb
```
